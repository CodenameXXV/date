📓 Суперзошит
Безкоштовний інтерактивний шкільний зошит онлайн для вчителів та учнів

Про проєкт
Суперзошит — це веб-застосунок, що імітує шкільний зошит у браузері. Не потребує реєстрації, встановлення та підключення до Інтернету після першого завантаження. Весь зошит — єдиний HTML-файл, що працює на комп'ютері, планшеті та смартфоні.

✨ Можливості
✏️ Текст та форматування

Жирний, підкреслений, закреслений текст — виділіть і натисніть кнопку або гарячу клавішу
Надрядковий і підрядковий — для ступенів, хімічних формул та індексів
Вирівнювання — по лівому краю, по центру, по правому краю
Колір тексту — синій, чорний, червоний, зелений
Нумерований список — автоматичне форматування
Вставка посилань — гіперпосилання прямо у тексті зошита

🖊️ Малювання від руки

Олівець — вільне малювання рукою; довге натискання змінює товщину лінії
Стирачка — стирає намальоване; довге натискання змінює розмір
Лінія — рівна лінія від точки до точки
Фігури — 12 готових фігур: прямокутник, заокруглений прямокутник, коло, еліпс, трикутник, прямокутний трикутник, ромб, зірка, серце, стрілка вправо, двостороння стрілка, шестикутник

🖼️ Зображення

Вставка зображень — з файлів або через Ctrl+V з буфера обміну
Переміщення та масштабування — перетяганням і за ріг
OCR — розпізнавання тексту — натисніть синю кнопку на вставленому зображенні; AI розпізнає текст і вставить його в зошит (підтримка UA та EN)
Обрізка — натисніть ✂️ на зображенні, виділіть область і збережіть

📌 Елементи на сторінці

Стікери — жовті плавучі нотатки; можна переміщувати, редагувати та видаляти
Довільний текст (фрітекст) — плавучий текстовий блок поза основним зошитом; переміщується, масштабується
QR-код — введіть посилання або текст, отримайте QR прямо на сторінці
Таймер — зворотний відлік, що перетягується; зупинка, скидання, сигнал завершення
Таблиці — гнучкі плавучі таблиці з можливістю додавати/видаляти рядки й стовпці, змінювати розмір

🎙️ Голос та математика

Голосове введення — натисніть мікрофон і диктуйте текст; індикатор показує активність
Математичні підказки — напишіть вираз, наприклад 2+2=, натисніть пробіл — з'явиться підказка з результатом; Enter — вставити
Панель математичних символів — дроби, корені, степені, грецькі літери та наукові символи одним кліком
Панель спецсимволів — розширений набір спеціальних знаків (Ω)

🤖 Штучний інтелект (AI)

AI-помічник — натисніть ★ на панелі; введіть або продиктуйте запит («Склади 5 речень англійською», «Поясни теорему Піфагора») — AI згенерує текст прямо в зошит
Перевірка орфографії та граматики — виділіть текст → кнопка ✓ у контекстному меню; AI виправить помилки і покаже зміни

⚙️ Налаштування зошита

Дата — вибір через календар; заголовок оновлюється автоматично; підтримка числового та словесного формату
Тип роботи — Домашня / Класна / Самостійна / Практична / Лабораторна / Творча / Контрольна / Тест / Диктант / Діагностувальна
Шрифт — «Друк» (Georgia курсив) або «Пропис» (рукописний шрифт Propysy)
Фон аркуша — лінійка, клітинка, коса лінія або чистий аркуш
Мова інтерфейсу — 🇺🇦 Українська / 🇬🇧 English / 🇩🇪 Deutsch

📄 Сторінки та навігація

Кілька сторінок — додавайте, видаляйте, перегортайте
Копіювати сторінку — дублює поточний аркуш у буфер обміну як зображення
Скачати сторінку — зберігає аркуш як PNG на пристрій
Скасувати / Повторити — Ctrl+Z / Ctrl+Y або кнопки на панелі

💾 Збереження та експорт

Автозбереження — усі дані зберігаються у localStorage автоматично; відкривайте наступного дня без втрат
Файл зошита — збережіть увесь зошит у файл .notebook (💾 ліворуч); відновіть з файлу будь-коли
Резервна копія — окрема кнопка експорту для надійності

🌐 Спільний доступ у реальному часі

Онлайн-редагування — натисніть 👥 на панелі; вчитель створює сесію і отримує код; учні вводять код і приєднуються до зошита
Синхронізація — зміни відображаються на всіх пристроях у реальному часі
Курсори учасників — видно, хто де знаходиться
Управління учнями — вчитель може дозволити/заборонити редагування, відключити учня

📱 Встановлення як застосунок (PWA)

Встановлюється на робочий стіл комп'ютера або домашній екран смартфона
Працює офлайн після першого відкриття
Банер встановлення на мобільних; кнопка у панелі на десктопі


🛠️ Технічний стек

Чистий HTML / CSS / JavaScript — без фреймворків, без залежностей
Єдиний self-contained файл index.html
Service Worker для офлайн-роботи та PWA
localStorage для автозбереження
WebRTC / WebSocket для спільного редагування
Web Speech API для голосового введення
Canvas API для малювання та експорту сторінок
